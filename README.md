# 📂 Excel2DicomSR

Automatic conversion of Excel values into DICOM SR and C-STORE to SR mapping server for automated population of algorithmic data in reporting systems. 

---

## 🚀 Getting Started

To use this tool, follow these two simple steps:

### 1. Download the Package
Click the green **`<> Code`** button at the top of this page and select **[Download ZIP](https://github.com/Mureozwaro/Excel2DicomSR/archive/refs/heads/main.zip)**.

### 2. Setup & Execution
* **Extract** the contents of the ZIP file to a folder on your computer.
* Open the extracted folder and view **`InstructionsForUse.PDF`** for the full setup guide and workflow details.

---

## 📦 What's Included?

| File/Folder | Description |
| :--- | :--- |
| **`CTA_Cardiac.xlsm`** | The functional workbook for user input, XML mapping, conversion to DICOM SR, and C-STORE to SR server. |
| **`LaunchCTA.vbs`** | The script to launch the workbook as external link in PACS. |
| **`Bin/`** | Contains the necessary binaries (DCMTK EXEs/DLLs). |
| **`Output/`** | Placeholder folder for processed data (XML, DICOM). |
| **`Submitted/`** | Placeholder folder for submitted workbooks. |
| **`InstructionsForUse.pdf`** | Prerequisites, setup and workflow instructions. |

> **Note:** Some browsers or antivirus software may flag the `.exe` and `.dll` files within the Bin folder as "unrecognized." This is normal for standalone toolkits. Ensure you extract the folder entirely for the script to function correctly.
